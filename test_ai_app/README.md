# 测试AI应用 (Test AI App)

面向测试业务场景的大模型应用，实现 RAG 检索增强、任务分流、流程编排与工具调用集成。

## 功能特性

- ✅ RAG 检索增强 - 知识库问答
- ✅ 任务分流 - 智能意图识别
- ✅ 内容生成 - 自动生成测试用例
- ✅ 流程编排 - 工作流自动调度
- ✅ 工具调用 - 浏览器自动化、API调用、文件操作
- ✅ 自动化验证 - 测试执行与结果验证
- ✅ 结果回传 - WebSocket/回调通知

## API入口

| 路由 | 方法 | 功能 |
|------|------|------|
| `/api/test-ai/task/` | POST | **主入口** - 完整业务流程 |
| `/api/test-ai/analyze/` | POST | 意图分析 |
| `/api/test-ai/retrieve/` | POST | RAG检索 |
| `/api/test-ai/generate/` | POST | 测试用例生成 |
| `/api/test-ai/execute/` | POST | 测试执行 |
| `/api/test-ai/tool/` | POST | 工具调用 |
| `/api/test-ai/tools/` | GET | 工具列表 |
| `/api/test-ai/history/` | GET | 任务历史 |

## 使用示例

### 完整业务流程

```bash

  -H "Content-Type: application/json" \
  -d '{
    "user_input": "帮我为一个登录页面生成测试用例，并执行自动化测试"
```
